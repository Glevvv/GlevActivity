Object-Oriented Programming Activity

 

 

 

 

 

 

Title:

Build Your Own Mini Zoo Simulator

(Inheritance & Polymorphism)

 

 

 

 

Objective:

Create your own version of a zoo simulator using inheritance and polymorphism, with unique animals and features.

 

 

 

 

 

Instructions:

1. Base Class:
Create a base class Animal with the following properties and methods:

o    name (String)

o    type (String)

o    age (int)

o    Method: makeSound() - Prints a generic message like "This animal makes a sound."

2. Derived Classes:

o    Students must create at least three different subclasses (e.g., Bird, Mammal, Reptile) and add unique properties/methods.

§  Example: Bird might have wingSpan, and a method fly().

§  Example: Mammal might have furColor and a method run().

3. Overriding Methods:

o    Override the makeSound() method in each subclass to provide a unique implementation for the specific animal.

4. Polymorphism:

o    Create a driver class (ZooTest) where students:

§  Instantiate their animals.

§  Store them in an array or ArrayList of Animal.

§  Use a loop to invoke makeSound() and any other overridden methods.

 

5. Dynamic Requirement:
To ensure individuality:

o    Each student must create unique animals, giving them creative names, types, and properties (e.g., Dragon, Penguin, Lion).

o    Include a user input section where students dynamically add new animals to their zoo during runtime.
